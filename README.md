# MC 1.12.2 Forge MOD Launcher

Minecraft 1.12.2 + Forge `14.23.5.2859` 用の自作ランチャー補助ツールです。

## ダウンロード

配布用ZIPはこちらからダウンロードできます。

- [MC1122-JP-Modpack-Launcher.zip](dist/MC1122-JP-Modpack-Launcher.zip)

## 使い方

ZIPを展開して、Windowsでは `run.bat`、macOS/Linuxでは `run.sh` を実行してください。

詳しい説明はZIP内の `README.md` を読んでください。

## 注意

このリポジトリが private の場合、権限がある人だけがダウンロードできます。友人にURLだけで配る場合は、GitHubの repository settings から public に変更してください。
